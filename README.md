# Procesamiento-en-batch_analisis_SaberPRO
# Análisis de datos con Apache Spark
Este proyecto realiza un análisis de datos utilizando Apache Spark sobre un conjunto de datos relacionado con los resultados del examen Saber Pro.
# Objetivo
Analizar la relación entre factores socioeconómicos y el rendimiento académico de los estudiantes mediante el procesamiento de datos en batch.
# Tecnologías utilizadas
Apache Spark
Python (PySpark)
Hadoop (HDFS)
# Procesos realizados
Carga de datos desde HDFS
Limpieza de datos (eliminación de nulos y duplicados)
Transformación de datos
Análisis exploratorio
# Instrucciones de ejecución
Tener configurado Hadoop y Spark
Subir el dataset al HDFS
python3 tarea4.py
Se colocó tarea 4 ya que en la anterior actividad el archivo se llamaba tarea3.
